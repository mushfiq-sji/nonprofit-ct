## 🛠️ Technology Stack

| Layer | Technology |
|-------|------------|
| **Frontend** | React 18 + TypeScript + Tailwind CSS + shadcn/ui |
| **Backend** | Supabase (PostgreSQL + Edge Functions + Auth) |
| **Deployment** | Lovable.dev or Self-hosted |
| **AI** | Lovable AI (included) or OpenAI/Anthropic |
